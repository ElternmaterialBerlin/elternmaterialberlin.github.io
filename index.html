Elternmaterial Berlin

Diese Seite sammelt material für Eltern von Kindern an Berliner Schulen. Schwerpunkt ist aktuell die Elternarbeit in Gremien, aber zukünftig sind auch Lernmaterialien, Medientipps etc. denkbar. Nebenher können weitere Web-Werkzeuge u.ä. gesammelt werden, z.B. für Umfragen.

Inhalte dieser Webseite

Die Inhalte sind jeweils direkt über die Webseite und über Git-Repositories verfügbar. Das ermöglicht bei Interesse eine einfache Mitarbeit (auch ohne Kenntnisse von Git).

  • Gremiendoks: Kontaktlisten, Vorlagen für Protokolle (u.a. Wahlprotokoll für die GEV) etc. (Quelltexte bei bei GitHub, bei GitLab)

Mitmachen

Wer Verbesserungen für die vorhandenen Materialien oder gänzlich neue hat, kann Frank Fuhlbrück kontaktieren. Bei einmaligen Vorschlägen kann das gerne direkt per Mail erfolgen. Längerfristige Interessenten werden in eine der Gruppen bei GitHub/GitLab aufgenommen.

GitHub oder GitLab?

Egal. Beide Seiten werden synchron gehalten. Wer nur etwas herunterladen möchte findet auf beiden dieselben Inhalte. Beiträge sind auch über beide Plattformen möglich.
